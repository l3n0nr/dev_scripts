# $ = símbolo do shell prompt

==========================
TRABALHANDO COM DIRETORIOS
--------------------------

# ls = list, lista os arquivos do diretório atual
	Exemplo: ~$ ls

# ls -a = lista os arquivos do diretório atual incluindo os que estão ocultos
	Exemplo: ~$ ls -a

# ls -l = lista os arquivos do diretório atual em formato longo (permissões - links - dono - grupo - tamanho - última modificação)
	Exemplo: ~$ ls -l

# ls -t = lista os arquivos do diretório atual ordenando-os pela data de modificação
	Exemplo: ~$ ls -t

# pwd = print working directory, exibe o diretório em que o usuário está atualmente
	Exemplo: ~$ pwd

# cd = change directory, muda de diretório
	Exemplo: ~$ cd /Documents

# cd .. = volta para o diretório anterior
	Exemplo: ~$ cd ..

# mkdir = make directory, cria uma pasta/diretório com o nome especificado
	Exemplo: ~$ mkdir MeuNovoDiretorio

========================
TRABALHANDO COM ARQUIVOS
------------------------

# touch = cria um arquivo no diretório atual com o nome especificado
	Exemplo: ~$ touch MeuNovoArquivo

# cp /dir/arquivo  /novodir/ = copia um arquivo de um diretório para outro
	Exemplo: ~$ cp /user/Documents/Aulas.pdf /user/MeuNovoDiretorio/

# mv /dir/arquivo  /novodir/ = move um arquivo de um diretório para outro
	Exemplo: ~$ mv /user/Documents/Prova.pdf /user/MeuNovoDiretorio/

# cp m*.txt /novodir = copia todos os arquivos que começam com 'm' e terminam em '.txt' para o novo diretório ( '*' é chamado de wildcard)

# mv nomeantigo novonome = renomeia o arquivo passado como primeiro argumento com o nome passado no segundo argumento

# cat arquivo = exibe o conteúdo do arquivo na linha de comando

# cat arquivo1 > arquivo2 = copia o conteúdo do arquivo1 para dentro do arquivo2, apagando o conteúdo do arquivo2

# cat arquivo1 >> arquivo2 = copia o conteúdo do arquivo1 para dentro do arquivo2, mantendo o conteúdo do arquivo2

# sort arquivo.txt = organize e mostra o conteúdo do arquivo em ordem alfabética na tela

# cat arquivo.txt | sort > arquivo_organizado.txt = Organiza o conteúdo do arquivo.txt em ordem alfabética e escreve-o em arquivo_organizado.txt

# uniq arquivo.txt = exibe o conteúdo do arquivo eliminando dados repetidas

# sort arquivo.txt | uniq = mostra o conteúdo organizado em ordem alfabética e sem dados repetidos. Copia para outro arquivo adicionando '> arq'

# grep xyx arquivo.txt = global regular expression print, procura dados que possuam a expressão 'xyz' no arquivo e exibe-os

# sed 's/snow/rain/' forests.txt = stream editor, 's/' é sempre usado com esse comando. 
##O argumento 'snow' é o que deve substituido, e 'rain' é o que substituirá 'snow'. Dessa forma substitui só a primeira ocorrência em uma linha.

# sed 's/snow/rain/g' forests.txt = mesmo que o comando anterior, porém ao adicionar o argumento 'g' ao final é feita uma substituição global. 
##Ou seja, todas as ocorrências de cada linha.
