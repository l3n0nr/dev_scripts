	environment block to small

	one  or more block group descritpyion checksums are invalid
”

Solução 1:

//corrigindo problemas no disco, atráves de verificação de setores

sudo fsck /dev/sda3 -c -p -v -f

Solução 2:

	sudo rm -rf /boot/grub2/grubenv

	sudo grub2-editenv /boot/grub2/grubenv create

	Link: <https://crimsonlinux.wordpress.com/2012/08/13/error-environment-block-too-small/>

Solução 3:

cd /boot/grub
rm grubenv
grub-editenv grubenv create
grub-editenv grubenv set default=0
grub-editenv grubenv list
