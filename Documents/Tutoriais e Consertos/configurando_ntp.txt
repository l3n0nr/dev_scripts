#Este comando irá sincronizar a data e hora do computador com um servidor NTP na Internet.
#Normalmente as distribuições já possuem esse comando habilitado.

sudo ntdate pool.ntp.org

#É possivel definir um rotina no crontab da distribuição, para a tarefa ser executada automaticamente

sudo crontab -e

#Após abrir o arquivo, vá até o final para definir a rotina 

#Iremos utilizar o exemplo para definir a rotina "m h  dom mon dow   command"

----------------------------------------------------------------------------------
m = minuto que o comando deve ser executado o comando;
h = hora que o comando deve ser executado;
dom = dia do mes;
mon = mes
mon = dia da semana;

command = comando que deve ser executado, no caso para sincronizar data e hora
----------------------------------------------------------------------------------

#Exemplo: caso eu queria que a rotina seja executada as 22h todos dias, basta definir esta instrução

00 22 * * * ntpdate pool.ntp.org

# No exemplo o '*' serve para apontar que o comando deverá ser realizado em qualquer dia do mes, mes e dia da semana
