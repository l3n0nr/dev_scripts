Aparece esse erro ao tentar subir ao arquivo no Github 

	"fatal: Authentication failed"

Possivel solução

-git remote set-url origin <ssh url>

	<ssh url> = nome do repositorio alvo
	
Depois tentar subir novamente

-git push
